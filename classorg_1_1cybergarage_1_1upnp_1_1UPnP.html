◆ loadDefaultXMLParser()

static Parser org.cybergarage.upnp.UPnP.loadDefaultXMLParser ( )
inlinestaticprivate

This method loads the default XML Parser using the following behavior:

  • First if present loads the parsers specified by the system property UPnP#XML_CLASS_PROPERTTY
  • Second by a fall-back technique, it tries to load the XMLParser from one
    of the following classes: JaxpParser, kXML2Parser, XercesParser
Returns
Parser which has been loaded successuflly or null otherwise
Since
1.8.0
